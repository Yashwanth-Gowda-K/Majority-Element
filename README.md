# Majority-Element


🔍 Problem
> Given an array `nums` of size `n`, return the **majority element**.  
> The majority element is the one that appears more than ⌊n / 2⌋ times.  
> You may assume that the majority element **always exists**.

 🧪 Example
```python
Input: nums = [2, 2, 1, 1, 1, 2, 2]
Output: 2
